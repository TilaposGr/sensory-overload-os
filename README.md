# sensory-overload-os

A meme-os config (for now) based on fedora linux 44, using the style of the 2026 dropper game sensory overload by pancake studios


#Installation guide

1.You need to have a fedora linux installation with kde installed

2.download the "utterly round" kde plasma theme by Himdeck (https://store.kde.org/p/1901768) as it will be used for the plasma style

3. apply the plasma style
   
4.Install the rpm package for the darkly theme by bali10050 https://github.com/Bali10050/Darkly, as it is needed for the application style

5.Apply that, then go to settings and uncheck all the options in the shadows tab an d check the "no outline" option

6.Right click the panel and click "show panel configuration"

7.Put it on the top of the screen, make it opaque, make  size 34, replace the application launcher with only icons with the option on the button while pressing "show alternatives", which ypou should press to configure and make it only text

8.Add a spacer between the application launcher and the one above from everything else

9.make another panel, specifically with the default panel preset

10.Make size 68, make it fill content, and only keep th3e icons only applications launcher and make translucent

11.Download the two folders of the repository using the download option as a zip

12.Extract the zip archive

13.Right click on the application launcher button and click "show panel configuration"

14.Select the "s-o.png" file from the "misc" folder

15.Apply and exit

16.Right click on the "soswall.png" file inside "wallapers" and select the "set as wallpaper" option and select "both" from the sub menu
