# sensory-overload-os

A meme-os config (for now) based on fedora linux 44, using the style of the 2026 dropper game sensory overload by pancake studios. Uses the "utterly round" kde plasma theme by Himdeck (https://store.kde.org/p/1901768) as it is 
used for the plasma style and the darkly theme by bali10050 https://github.com/Bali10050/Darkly, as it is needed for the application style


## Installation guide

1.You need to have a fedora linux installation with kde installed

2.Download the files from the repository using the download option as a zip

3.Extract the zip archive **DO NOT RENAME IT**
   
4.Go to your terminal of choice

5.Type "cd ~/Downloads/sensory-overload-os-main" in order to go to the downloaded folder

6.Type "./apply-config.sh" in order to execute it

7.Accept if it asks for permission and wait

8.Congratulations, the modifications should now be applied onto your system!!!
